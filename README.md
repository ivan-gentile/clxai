# CLXAI: On the Properties of Feature Attribution for Supervised Contrastive Learning

This repository contains the code for reproducing the experiments presented in:

> **On the Properties of Feature Attribution for Supervised Contrastive Learning**
> Leonardo Arrighi, Julia Eva Belloni, Aurélie Gallet, Ivan Gentile, Matteo Lippi, Marco Zullich

## Overview

We empirically show that neural networks for image classification trained with Supervised Contrastive Learning (SCL) produce higher-quality feature attribution explanations than Cross-Entropy (CE) training, with regard to **faithfulness**, **complexity**, and **continuity**.

We compare three training paradigms:
- **Cross-Entropy (CE)**: standard classification loss
- **Supervised Contrastive Loss (SCL)** ([Khosla et al., 2020](https://arxiv.org/abs/2004.11362))
- **Triplet Loss (TL)** ([Schroff et al., 2015](https://arxiv.org/abs/1503.03832))

We evaluate explanations generated by **Grad-CAM** and **Eigen-CAM** on five quality facets:
faithfulness, coherence, continuity, contrastivity, and complexity.

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Download datasets

**CIFAR10** is automatically downloaded by torchvision.

**ImageNet-S50** requires:
1. Download ImageNet (or the 50-class subset) in HuggingFace Arrow format to `data/imagenet-1k/`.
2. For coherence evaluation, download segmentation masks from [ImageNet-S](https://github.com/LUSSeg/ImageNet-S).

## Training

### CIFAR10 (ResNet-18)

```bash
# Cross-Entropy
python -m src.training.train_ce --epochs 200 --lr 0.1 --batch_size 128 --seed 0

# Supervised Contrastive Loss
python -m src.training.train_scl --loss supcon --epochs 500 --lr 0.5 --batch_size 256 --seed 0

# Triplet Loss
python -m src.training.train_scl --loss triplet --epochs 500 --lr 0.1 --batch_size 256 --seed 0
```

### ImageNet-S50 (ResNet-50)

```bash
# Cross-Entropy
python -m src.training.train_ce_imagenet50 --data_dir data/imagenet-1k --epochs 160 --seed 0

# Supervised Contrastive Loss
python -m src.training.train_scl_imagenet50 --data_dir data/imagenet-1k --loss supcon --epochs 160 --seed 0

# Triplet Loss
python -m src.training.train_scl_imagenet50 --data_dir data/imagenet-1k --loss triplet --epochs 160 --seed 0
```

All models are trained from scratch (no pretrained weights). We train on **5 different seeds** (0-4) per configuration.

## Evaluation

### Faithfulness, Continuity, Contrastivity, Complexity

```bash
# CIFAR10
python -m src.evaluation.xai_evaluation \
    --model_type ce \
    --checkpoint path/to/best_model.pt \
    --dataset cifar10 \
    --metrics all

# ImageNet-S50
python -m src.evaluation.faithfulness_imagenet50 \
    --checkpoint path/to/best_model.pt \
    --model_type ce \
    --data_dir data/imagenet-1k \
    --metrics all
```

### Coherence (ImageNet-S50 only)

```bash
python -m src.evaluation.coherence_imagenet50 \
    --checkpoint path/to/best_model.pt \
    --model_type ce \
    --data_dir data/imagenet-s50/ImageNetS50 \
    --output_dir results/coherence/ce
```

### Saliency Visualization

```bash
python scripts/visualize_saliency.py \
    --data_dir data/imagenet-s50/ImageNetS50 \
    --checkpoints CE:ce:path/to/ce_model.pt SCL:encoder:path/to/scl_model.pt:path/to/linear.pt \
    --num_images 5 --output_dir results/saliency_visualizations
```

### Results Comparison

```bash
python scripts/compare_results.py \
    --results_dirs results/xai_eval/ce results/xai_eval/scl results/xai_eval/triplet \
    --model_names CE SCL TL \
    --output_dir results/analysis
```
